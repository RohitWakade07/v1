# Week 01 - Linux Environment and Filesystem Navigation

## Conceptual Focus
Set up the Linux working environment, use the shell as the primary developer tool, and perform fundamental file operations.

## Core Track (graded)
- Terminal exploration and shell basics: prompt structure, environment variables, identity commands.
- Filesystem navigation drill: pwd, cd, ls with flags; absolute vs relative paths.
- File and directory operations: touch, mkdir, cp, mv, rm; inspect content and metadata.
- Command documentation: man, --help, which.

## Exploration Track (optional)
- Customize the shell prompt with PS1 and colors.
- Advanced ls and find usage; permissions and file types.
- Symbolic links and PATH management.

## Mini Project (5 marks)
Developer workspace setup:
- Create the course directory tree under ~/eep-software.
- Add README.md to each week folder using nano.
- Add at least two aliases to .bashrc.
- Produce workspace-report.txt with the tree listing and .bashrc additions.
