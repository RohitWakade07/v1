# Week 03 - Shell Scripting and Automation

## Conceptual Focus
Move from one-liners to reusable scripts that take input, branch on conditions, and loop over data.

## Core Track (graded)
- Variables, arguments, user input, and shebang usage.
- Conditionals with if/elif/else and [[ ... ]].
- Loops with for and while; batch processing.
- Safe bulk operations with dry-run habits.

## Exploration Track (optional)
- case statements and pattern matching.
- Logging and timestamped output.
- Filename collision handling and cron jobs.
- Script hardening with set -euo pipefail.

## Mini Project (5 marks)
Automated File Organizer:
- organize.sh sorts files into Documents/, Images/, Code/, Other/.
- Validate input path and print a summary of moved files.
