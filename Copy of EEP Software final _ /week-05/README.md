# Week 05 - Collaborative Development with GitHub

## Conceptual Focus
Extend Git to team workflows: remotes, pull requests, and conflict resolution.

## Core Track (graded)
- Remote setup and authentication.
- Push, pull, fetch workflows across clones.
- Merge conflict resolution.
- Pull request basics.

## Exploration Track (optional)
- SSH keys, review etiquette, branch protection.
- Rebasing vs merging, long-term forks.

## Mini Project (5 marks)
Team Repository Recovery Challenge:
- Resolve conflicts in a shared repo, review a PR, and document teamwork in TEAMWORK.md.
