# Week 02 - Command-Line Tools and Text Processing

## Conceptual Focus
Treat text as the universal data format and use small, composable tools to extract information from data.

## Core Track (graded)
- File searching with find and locate.
- Text filtering with grep, sort, uniq, wc.
- Pipelines and I/O redirection.
- Server log exploration with one-line pipelines.

## Exploration Track (optional)
- Extended regex with grep -E.
- Text transformation with sed and awk.
- Column tools: cut, paste, tr.

## Mini Project (5 marks)
Command-Line Log Analyzer:
- Build a shell script of pipelines to analyze a log file.
- Report: top IPs, top URLs, status distribution, total requests.
- Save outputs to report.txt.
