# Week 04 - Version Control with Git

## Conceptual Focus
Treat code as a tracked history using commits, branches, and diffs.

## Core Track (graded)
- Initialize repositories and understand staging.
- Commit workflow and change tracking.
- Explore history with log, diff, show.
- Branching and merging basics.

## Exploration Track (optional)
- Interactive staging and history rewriting.
- Tagging, conflicts, blame, bisect.

## Mini Project (5 marks)
Local Repository Recovery Challenge:
- Clean a broken repo, fix history, and document all Git commands in RECOVERY.md.
