# Week 09 - Linear vs Inverted Index and Frequency Counting

## Conceptual Focus
Build and compare naive search with an inverted index, then add frequency-based ranking.

## Core Track (graded)
- Linear search across documents.
- Build an inverted index and persist it.
- Compare runtime between linear and index lookup.
- Frequency counting per document and basic ranking.

## Exploration Track (optional)
- Stopword removal, TF-IDF, positional index.
- Boolean queries, fuzzy matching, binary search on vocabulary.

## Mini Project (5 marks)
Inverted Index Builder and Lookup Tool:
- build_index.py writes index.json with frequencies.
- lookup.py loads index.json and ranks results.
- Include timing comparison in README.md.
